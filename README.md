# Basketball Stats Tracker

Mobile-first PWA-style React app for tracking basketball stats (localStorage backed).

Quick start:

1. Install dependencies:

```bash
cd /Users/brandontaylor/code/claude/basketball_tracker
npm install
```

2. Run dev server:

```bash
npm run dev
```

This scaffold includes:
- React + Vite
- Tailwind CSS
- Lucide icons
- LocalStorage helpers

Files of interest:
- `src/App.jsx` - main app and views
- `src/utils/storage.js` - localStorage helpers
- `src/utils/calculations.js` - stat calculations
